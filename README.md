# Cinematech 

## Getting startd 

### Prerequisites

1. Frontend

```Bash
npm install --prefix ./frontend
```
 
2. Backend 

List des packages django (pip install) : 
- djangorestframework
- Pillow
- django-cors-headers


### Usage 

1. Frontend 

```Bash
npm start --prefix ./frontend
```

2. Backend

```Bash
./backend/.venv/bin/python ./backend/manage.py runserver
```
